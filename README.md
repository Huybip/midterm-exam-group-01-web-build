# Bread-Shop

Nhằm đáp ứng nhu cầu mua bán của khách hàng một cách nhanh chóng và hiệu quả tránh mất thời gian đi lại và chọn lựa sản phẩm.
Thông tin được cập nhất thường xuyên để đáp ứng nhu cầu của người dùng

# Cách chạy chương trình

Bước 1: php artisan migrate

Bước 2: php artisan db:seed

Bước 3: chạy thêm npm run-dev để chạy phần css và js

Bước 4: php artisan serve: chạy chương trình chính
