HƯỚNG DẪN SỬ DỤNG THƯ MỤC ẢNH BÁNH MÌ
=====================================

1. CÁCH THÊM ẢNH:
   - Copy ảnh bánh mì vào thư mục này (public/images/breads/)
   - Định dạng hỗ trợ: JPG, JPEG, PNG, GIF, SVG
   - Tên file nên đặt rõ ràng, ví dụ: banh-mi-pate.jpg

2. CÁCH SỬ DỤNG:
   - Vào Admin → Quản lý bánh mì
   - Khi thêm/sửa bánh mì, chọn ảnh từ dropdown "Chọn ảnh từ thư mục"
   - Hoặc upload ảnh mới (sẽ lưu vào storage)

3. LƯU Ý:
   - Ảnh trong thư mục này sẽ hiển thị trực tiếp trên website
   - Không cần chạy lệnh storage:link cho ảnh trong thư mục này
   - Ảnh upload qua form sẽ lưu vào storage/app/public/breads/
